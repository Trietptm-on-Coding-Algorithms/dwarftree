# dwarftree

A visual, tree-based, DWARF explorer. It uses [pyelftools](https://github.com/eliben/pyelftools).

## Dependencies

* Python 3
* pyelftools
  * `sudo pip3 install pyelftools`
* PyGObject / PyGI
  * Debian/Ubuntu: `python3-gi` package
  * Fedora: `pygobject3` package
